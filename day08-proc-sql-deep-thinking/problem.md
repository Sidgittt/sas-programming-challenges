Day 8 – PROC SQL Deep Thinking Challenge

Objective: Strengthen set-based thinking using PROC SQL.

Focus Areas:

• GROUP BY logic  
• HAVING clause  
• Self-joins  
• Correlated subqueries  
• Avoiding unnecessary sorting  
• Thinking in sets instead of row-wise processing  

Tasks:

1. Calculate average VALUE per subject using GROUP BY  
2. Select subjects whose average VALUE is greater than overall average (HAVING)  
3. Perform self-join to compare current visit VALUE with previous visit  
4. Use correlated subquery to identify maximum VALUE per subject
