# SAS Programming Challenges

This repository contains SAS programming challenges designed to improve DATA step logic and clinical data programming skills.

Topics Covered

• Duplicate removal using PROC SORT, PROC SQL, and HASH  
• FIRST./LAST. BY-group processing  
• Running calculations  
• RETAIN vs SUM statement  
• Date derivations  
• Array processing  
• Clinical derivations like Baseline Flag, LOCF, Change from Baseline
